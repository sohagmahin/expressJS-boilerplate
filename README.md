# ExpressJS-boilerplate

ExpressJS boilerplate with basic auth functionality.

### Signup schema

```
{ 
  "name": "sohag", 
  "username": "sohagmahin", 
  "password": "123456" 
 }
```

### Login Schema

```
{ 
  "username": "sohagmahin", 
  "password": "123456" 
 }
```
